## What is this mod about?

This mod aims to be the best rebalance and content mod for all types of players! Once you sink into the game, you won’t want to return to the original. 
The following changes are eihter current or future changes.

## 🔧 What has changed in this mod?

| Feature                          | Original                         | My Mod                                                      |
|---------------------------------|---------------------------------|-------------------------------------------------------------|
| Boss Raid              | Somewhat Not Great Rewards              | More Rewarding And Elite Variants For All Bosses                                             |
| Mission                     | Days Were Too Easy           | Somewhat Difficult But Balanced, While Also Adding Variation And Scrapped Mission Types                             |
| Weapons                   | Super Unbalanced And Expensive/Too Cheap | Prices Rebalanced And Rebalanced Guns                                     |
| Upgrades                   | Unbalanced Weapon Upgrading                          | Somewhat Restrictive Upgrades                                                   |
| Skills                   | Super Weak/Too Strong | Prices Rebalanced And Rebalanced Skill Upgrades And Pricing                                     |

| UI                          | Original                      | Enhanced Main Menu Look                                      |

## [1.3.3] – Latest  

### PC  
- Automatic cursor lock.  
- Widescreen support fixed (map arrows now display).  
- PC controls improved again.  
- Skip cutscenes with **Space**.  

### Characters  
- Armor upgrade cost with Vouchers reduced by **30–45%**.  
- Stat balance adjustments.  
- Fixed random ability soft-locks.  
- Fixed green outline bug.  

### Boss Raid  
- Elite bosses spawn more minions.  
- Boss HP rebalanced.  
- Normal Rugby helmet no longer matches Elite Rugby.  
- Elite boss icons added in-game and Coop.  
- Fixed missing boss icons in Coop Room.  
- Removed “Random” button (no multiplayer).  
- Boss fights now require a minimum day level.  

### Enemies  
- Enemies now drop ammo and HP (Endless mode: Crystals and Cash).  
- Shark Hunter projectiles now home on players.  

### Weapons  
- New **M16** icon and model (from Squad).  

### Game Modes  
#### Endless Mode  
- New map (reworked CoM Squad).  
- Difficulty increases the longer you survive.  
- Enemies drop Crystals and Cash.  

### Missions  
- Higher day levels give more rewards (Vouchers, Cash, Crystals).  
- Each mission type now has its own panel.  
- All modes drop Ammo and HP.  
- Old mission boxes removed (new system in place).  

### Maps  
- New Endless map (see #sneak-peaks).  
- Updated most textures for improved visuals.  

### Performance  
- Enemy spawn system overhauled → smoother gameplay, less stutter.  

### Daily Missions  
- Vouchers increased by **50%**.  
- Hard missions give **5x Cash**.  
- Fixed missing Lottery items on boot.  
- Higher days = more Vouchers and Cash.

## Installation Instructions

### Android

1. Download the latest APK from the [Releases](https://github.com/ShadowOsmium/COM-Zombies-2/releases/latest) page — file:   
   `Android.COM-Zombies-2.apk`
2. Allow installation from unknown sources on your Android device if needed.
3. Open the APK to install the game.

### PC (Windows)

1. Download the latest PC Release from the [Releases](https://github.com/ShadowOsmium/COM-Zombies-2/releases/latest) page — file:   
   `PC.CoM.Zombies.2.7z`
2. Extract the ZIP file to your preferred folder.
3. Run the executable inside to start the game.

## 🎮 Gameplay Controls

### Android

- Use touchscreen controls and on-screen buttons.  

### PC (Windows)

- **Move:** WASD  
- **Shoot:** Left Mouse Button  
- **Pause/Menu:** Esc key  
- **Ability:** Num 1
- **Skip Cutscene:** Space
- **Buy Ammo:** R
